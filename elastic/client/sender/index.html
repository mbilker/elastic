elastic::client

Module elastic::client::sender [−] [src]

Low-level http client details.

This module contains low-level implementation details for the high-level Client. Most of the types here aren't currently usable directly, but once the low-level API matures they should be extensible. Some notable types include:

  • Sender: a generic trait that can send a http request and return a response
  • NextParams: a generic trait that can fetch a set of parameters to associate with a request
  • SyncSender: a synchronous http client
  • AsyncSender: an asynchronous http client.
